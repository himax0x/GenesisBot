How to Use

  • BallCam is set to "space" key. To delete BallCam, open rlmarlbot/main.py and delete pid=self.pid on line 855.
  • Jump Force Simple is set to "u". To change, go to marlbot/nexto/bot.py, find "u" and replace it with another key.
  • Jump Force Double is set to "i". To change, go to marlbot/nexto/bot.py, find "i" and replace it with another key.
